Generated by /plan-ceo-review on 2026-03-19

## Vision

### 10x Check

A full workforce analytics dashboard with per-employee quarterly performance scoring,
trend lines over multiple quarters, automated bonus calculation with configurable formulas,
manager approval workflows, payroll integration, department comparison charts, and anomaly
detection. Flagged as ocean territory — not lake. Individual lake-sized pieces are captured
as accepted expansions below.

## Accepted Scope (added to this plan)

## Design Specifications (added by /plan-design-review)

### Information Architecture

```
  ┌──────────────────────────────────────────────────────────┐
  │  HEADER (.custom-header) — "Island Time" branding        │
  ├──────────────────────────────────────────────────────────┤
  │  CONTROLS (.form-card, dark gradient)                    │
  │    PRIMARY: [Q1 2026] [Q2 2026] [Q3 2026] [Q4 2026]     │
  │      active=btn-primary, inactive=btn-outline-light      │
  │    SECONDARY: Start [____] End [____] Employee [All ▼]   │
  │    [Search]                                              │
  ├──────────────────────────────────────────────────────────┤
  │  SUMMARY BAR (.report-summary, green gradient)           │
  │    Total Hours | Total Shifts | Employees | Date Range   │
  ├──────────────────────────────────────────────────────────┤
  │  ACTION BAR (right-aligned)                              │
  │    [Download CSV]  [Print]                               │
  ├──────────────────────────────────────────────────────────┤
  │  DEPARTMENT SECTION (bg-light bar, dept name + subtotal) │
  │    TABLE (.custom-table)                                 │
  │    Employee | Shifts | Total Hours | [View →]            │
  │    Sorted: hours descending within each department       │
  ├──────────────────────────────────────────────────────────┤
  │  FOOTER: Company logo                                    │
  └──────────────────────────────────────────────────────────┘
```

- No jumbotron — go straight from header to form card
- Drill-down: explicit [View →] link in last column (not clickable rows)
- "Back to Quarterly Report" breadcrumb on drill-down shift report page
- Default sort: total hours descending within each department section

### Visual Tokens (reuse from existing pages)

- `--primary-color: #3498db` — buttons, table header gradient
- `--secondary-color: #2ecc71` — summary bar gradient
- `--accent-color: #f39c12` — time badges, active sort indicator
- `--dark-color: #2c3e50` — form card gradient
- Border radius: 30px buttons, 15px cards, 10px tables
- Font: Segoe UI, Tahoma, Geneva, Verdana, sans-serif
- Sort indicators: ▲/▼ in --accent-color for active, ⇅ muted gray for unsorted
- Dept section headers: bg-light bar, dept name left, subtotal right

### Print Layout (@media print)

- Header: "Quarterly Employee Report — [Q1 2026] — Island Time by Island Millwork Inc"
- Show summary stats + full table
- Hide: nav header, form controls, gradients, CSV/Print buttons, drill-down arrows
- Black on white, no backgrounds, clean table borders

### Responsive

- Desktop (>992px): full layout, 4 quarter buttons in row
- Tablet (768-991): quarter buttons 2x2 grid, date inputs stack, .table-responsive
- Mobile (<768px): quarter buttons stack full-width, .table-responsive for table, summary 2x2 grid

### Accessibility

- `<caption>` on table: "Q1 2026 Employee Hours Summary"
- `aria-sort` on sortable `<th>` elements
- Quarter presets: `<button>` with `aria-pressed`
- CSV: `aria-label="Download quarterly report as CSV"`
- Drill-down: `aria-label="View shift details for [Employee Name]"`
- Touch targets: 44px minimum (Bootstrap default)
